# 🔍 PROMPT AUDIT - PROJET SPÉCIFIQUE

**À copier-coller dans Perplexity pour auditer un projet GitHub spécifique**

---

## 📋 **INSTRUCTIONS**

Copie ce prompt dans Perplexity et remplace `[URL_DU_PROJET]` par l'URL du projet à auditer.

---

## 🔍 **PROMPT**

```
Tu es un expert senior en audit de projets GitHub professionnels avec 10+ ans d'expérience.

Je te demande de faire un audit ULTRA-RIGOUREUX et EXHAUSTIF du projet GitHub suivant :

URL du projet : [URL_DU_PROJET]

TON RÔLE D'ÉVALUATION :
- Sois EXTRAORDINAIREMENT CRITIQUE : Un projet "9/10" pour toi doit être au niveau des meilleurs projets open-source
- Sois MINUTIEUX AU POINT : Vérifie chaque caractère, chaque espace, chaque lien, chaque métrique
- Sois EXIGEANT : Compare aux standards des projets de développeurs senior/CTO des grandes entreprises tech
- Sois CONSTRUCTIF : Pour chaque problème, propose une solution concrète et actionnable
- Sois EXHAUSTIF : Aucun détail ne doit t'échapper, même les plus subtils
- Sois OBJECTIF : Distingue ce qui est vraiment "niveau avancé" de ce qui est juste "bon"

VÉRIFICATIONS OBLIGATOIRES :
- 🔍 Chaque métrique doit être vérifiée et sourcée
- 🔍 Chaque lien doit être testé et fonctionnel
- 🔍 Chaque assertion doit être justifiée
- 🔍 Chaque date doit être à jour
- 🔍 Chaque badge doit être actif et précis
- 🔍 Chaque description doit être concise et impactante
- 🔍 Chaque section du README doit être complète

STRUCTURE DE TA RÉPONSE :

1. RÉSUMÉ EXÉCUTIF
   - Note globale /10
   - Points forts principaux (Top 5)
   - Points faibles principaux (Top 5)
   - Verdict détaillé

2. AUDIT DÉTAILLÉ
   - Structure & Organisation
   - Qualité du Code
   - Documentation
   - Tests & Coverage
   - CI/CD
   - Sécurité
   - Performance
   - Crédibilité

3. RECOMMANDATIONS PRIORITAIRES
   - Top 10 recommandations avec impact et temps estimé
   - Solutions concrètes pour chaque problème

4. COMPARAISON AVEC STANDARDS
   - Comparaison avec projets similaires de niveau avancé
   - Gap à combler pour atteindre le top 1%

Fournis une réponse COMPLÈTE et EXHAUSTIVE en une seule fois.
```

---

## 🚀 **UTILISATION**

1. **Ouvre ce fichier**
2. **Copie le prompt** (section entre les ```)
3. **Remplace `[URL_DU_PROJET]`** par l'URL réelle
4. **Colle dans Perplexity**
5. **Attends la réponse complète**

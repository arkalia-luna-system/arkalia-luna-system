## 🎯 **Une seule fois : Créer le repository sur GitHub**

1. **Va sur** : https://github.com/arkalia-luna-system
2. **Clique** : "New repository" (bouton vert)
3. **Nom** : `arkalia-luna-system` (même nom que l'organisation)
4. **✅ Public + ✅ Add README**
5. **Clique** : "Create repository"

## 📝 **À chaque fois que tu veux mettre à jour :**

### **Méthode 1 : Copier-Coller (Plus Simple)**
1. **Modifie** : `/Volumes/T7/github-profile-arkalia/README.md` (ce fichier local)
2. **Va sur** : https://github.com/arkalia-luna-system/arkalia-luna-system
3. **Clique** : README.md → "Edit this file" (crayon)
4. **Sélectionne tout + supprime**
5. **Colle** : le contenu de ton fichier local
6. **Commit** : "update profile"

### **Méthode 2 : Script Automatique** 
1. **Modifie** : `/Volumes/T7/github-profile-arkalia/README.md` (ce fichier local)
2. **Lance** : `./scripts/update-all.sh` puis `git add . && git commit -m "update" && git push`
3. **C'est tout !**

## 📁 **Ton dossier local :**
```
/Volumes/T7/github-profile-arkalia/
├── README.md              # TON PROFIL (à modifier)
├── INSTRUCTIONS-SIMPLE.md  # Ce guide
├── scripts/update-all.sh  # Script automatique
└── CONTRIBUTING.md        # Fichier pour GitHub
```

Ton profil sera visible sur **https://github.com/arkalia-luna-system** (page principale)

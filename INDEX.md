## 🚀 **UTILISATION RAPIDE**

### **Pour auditer un projet avec Perplexity**
1. Ouvre `prompts/PROMPT-AUDIT-PERPLEXITY.md`
2. Copie le contenu
3. Colle dans Perplexity
4. Ajoute l'URL du projet à auditer

### **Pour voir où tu en es**
```bash
cat statut/STATUT.md
```

### **Pour découvrir des problèmes**
```bash
cat guides/GUIDE-DECOUVRIR-PROBLEMES.md
cat guides/CHECKLIST-VIGILANCE.md
```

### **Pour voir les plans d'amélioration**
```bash
cat plans/PLAN-ACTION-1-MOIS.md
cat plans/PLAN-SUITE-ARKALIA-METRICS.md
```

### **Pour voir les résultats des audits**
```bash
cat audits/AUDIT-PERPLEXITY-REPONSE.md
cat audits/AUDIT-BBIA-RIGOUREUX.md
```
